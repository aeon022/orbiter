What if your entire CMS was a single file?

Orbiter stores content, media, users, and settings in one `.pod` file — a standard SQLite database. No cloud, no Docker, no monthly bill.

Just shipped v0.3.64:

📅 Calendar view — month grid showing all your scheduled, published, and draft entries
🔄 Cross-pod copy — export a pod as JSON, import it into another project
🖥️ Desktop auto-update — Electron app updates itself from GitHub
📊 Table field — mini-spreadsheet right in the entry editor

Built for Astro. MIT-licensed. Deploys to any shared hosting via FTP.
