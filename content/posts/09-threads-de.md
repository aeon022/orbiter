Was wäre, wenn dein ganzes CMS eine einzige Datei wäre?

Orbiter speichert Content, Medien, User und Settings in einer `.pod`-Datei — eine normale SQLite-Datenbank. Keine Cloud, kein Docker, keine monatliche Rechnung.

Gerade v0.3.64 veröffentlicht:

📅 Kalenderansicht — Monatsraster mit allen geplanten, veröffentlichten und Entwurfs-Einträgen
🔄 Cross-Pod Copy — Pod als JSON exportieren, in ein anderes Projekt importieren
🖥️ Desktop Auto-Update — Electron App updatet sich selbst über GitHub
📊 Tabellen-Feld — Mini-Spreadsheet direkt im Entry-Editor

Gebaut für Astro. MIT-lizenziert. Deployt auf jedes Shared Hosting via FTP.
