Appearance & Themes

Three themes, two colour schemes, and two distinct layout modes — including Space Station mode with its floating dock and mobile tab bar.

Themes

Switch themes in Settings → Interface. The preference is saved to localStorage and takes effect immediately — no reload needed.

Space Station mode

Space Station mode is a separate layout built for the Space theme. It replaces the left sidebar with a floating magnification dock similar to macOS, adds a HUD status bar at the top, and includes a sidebar with a live notes widget and to-do list.

Enable it in Settings → Interface → Layout → Station.

Floating dock

The dock sits at the bottom of the screen. Icons magnify on hover (macOS-style). Navigation, build trigger, tools, and theme toggle are all in the dock. On mobile, the dock collapses into a native-feeling bottom tab bar.

HUD panel

A collapsible right panel shows site stats (entry count, published count, last build time) and the scratchpad + to-do widget. Toggle it with the HUD button in the dock.

Glass page headers

Each page gets a frosted glass header bar with the page title and contextual action buttons (e.g. "Save settings" on the settings page, "New entry" on the entries list). This keeps the primary action always visible without scrolling.

Mobile layout

On screens narrower than 768 px, Station mode automatically switches to a bottom tab bar with five tabs: Dashboard, Entries, Media, Schema, and Tools. The dock magnification effect is disabled on touch devices.

Classic layout

The default (non-Station) layout uses a fixed left sidebar and full-width content area. It supports all three themes and both colour schemes.

Glassmorphism panels

All themes support a Glass surface mode — frosted panels with backdrop blur and animated gradient orbs in the background. Switch between Glass and Solid in Settings. Glass is the default.
